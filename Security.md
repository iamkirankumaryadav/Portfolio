| ------- | ------------------ |
| 5.1.x   | :white_check_mark: |
| 5.0.x   | :white_check_mark: |

## Reporting a Vulnerability

Website is monitored under the GitHub Security Policy, any kind of misuse or download of data will be tracked and reported.
